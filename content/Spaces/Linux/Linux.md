I've moved away from **Windows** in favor of linux! My daily driver is currently **EndeavorOS** (EOS), an arch-based distro, and I _love_ it. I do have some issues, however, like not having direct access to **Worldographer**... I have to spin up an **ubuntu VM** and run the java version.

I've gotten the application to launch with the java version on EOS, but there's a lot of screen flickering. I've experienced similar issues with other `AppImages`, too. I believe it has some to do with **Wayland** and my **NVIDIA** graphics card. My next upgrade ([[Wants]]) will be an AMD card to get away from those proprietary drivers...

I'm loving the customizability. Gaming has become a non-issue for the [[Games MOC|games]] that I play.
