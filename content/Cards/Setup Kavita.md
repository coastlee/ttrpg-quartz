# Set Up Kavita

[Kavita](https://www.kavitareader.com/#downloads-v1-linux) looks to be a great option for managing my blossoming TTRPG PDF library!

Might be worth checking out **Calibre**, too.

Got this kicked off on localhost today. Discovered an ssh application that temporarily gives your localhost a domain to connect from other applications. Made me really want to get it up and running on my server.

[[20240617]]

Got the last bit working on 2024-06-18! Really excited about it!

There are certainly other features and enhancements to make. Like making it available publicly when I'm not home. Might be worth writing a script that I can easily kick off to open the ports only when I'm not home. Then another to close them when I get home. Whatever I can do to make it easy.

A lot of my tinkering was a result of PDFs not having metadata and figuring out how Kavita structured and organized it. Eventually I found out what I needed online and on discord. So I've got the series all bundled up mostly how I think I'd like it. This opened up access to a bunch of D&D 3.5e books I gained access to when I first started playing with so-and-so at the university!
