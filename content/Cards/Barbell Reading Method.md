# Read & Implement Barbell Reading Method

[Article](https://zettelkasten.de/posts/barbell-method-reading/)

**srasu** in the [[Mythic Game Master Emulator]] discord server mentioned this for retaining important things from read books. It seems the concept is focused around reading a book twice; the first time, you read it swiftly, without skipping material. Mark up the book.

On your second read through, only read the marked sections. Then cross-reference to other notes, make drawings, connections. This should help you retain that information much better!
